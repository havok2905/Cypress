What is Cypress?

Cypress is a bare bones, no special features image gallery that is easy to configure and setup with a default theme. Cypress simply contains the functionality of selecting an image, displaying the result, and allowing you to navigate through content. Cypress contains minimal html elements to make custom styling a breeze for any designer. Also, the use of objects as the containers for our image data allows for Cypress to include images from any source.

How to Implement Cypress

Make sure you have a version of jQuery included in your application as well as the cypress.js file and any downloaded themes. 

<head>
	<title>Cypress Gallery</title>
	<script src="js/jquery.js"></script>
	<script src="js/cypress.js"></script>
	<link rel="stylesheet" type="text/css" href="css/theme.css"/>
</head>

To implement this plugin in your website first create a div tag with the id "gallery" where you want the gallery to appear. 

<div id="gallery"></div>

In a JavaScript file you want to now initialize the gallery. We do this by adding an object with our image information inside. Then we call a build method. It should be structured as such. 

$(document).ready(function()
{
	$("html").cypress({"images":[
		{
			"title": "Country Side",
			"desc": "Picture of rural Texas",
			"thumb": "img/gallery01_thumb.jpg",
			"normal": "img/gallery01.jpg",
		},
		{
			"title": "Airplane Shot",
			"desc": "Picture of lakes over Orlando Florida",
			"thumb": "img/gallery02_thumb.jpg",
			"normal": "img/gallery02.jpg",
		},
	]
	});
	$("html").cypress("build");
});

Style Guide

<div id="gallery">
	<div id="image-container" class="container">
		<img src="img/gallery01.jpg" alt="" id="gallery-image">
		<h2>Title</h2>
		<ul id="nav-list">
			<li>
				<a href="prev" id="#prev">prev</a>
			</li>
			<li>
				<a href="next" id="#next">next</a>
			</li>
		</ul>
		<a href="back" id="#back">back</a>
	<div/>
	<div id="gallery-container" class="container">
		<ul>
			<li>
				<a href="#" data-role=0>
					<img src="img/image.png" alt="The description"/>
				</a>
			</li>
		</ul>
	</div>
</div>
div#gallery
#gallery is the container for the entire gallery plugin.
div.container
.container is what houses both of the two content containers of this plugin; the displayed image as well as the image selection window. In the demo, it is what provides the grey borders.
div#gallery-container (also has the container class)
#gallery-container contains the list of images which you select.
div#image-container (also has the container class)
#image-container houses our image that we selected as well as some navigation controls.
div#image-container img#gallery-image
This is the larger version of the image you selected
div#image-container h2
This is the title of our selected image.
div#image-container ul#nav-list
This contains the links that allow us to move from image to image when the #image-container is visible.
div#image-container ul#nav-list li a#prev
This is our previous image link.
div#image-container ul#nav-list li a#next
This is our next image link.
div#image-container a#back
This link is what takes us back to the image selection window.
